# Octodo

[![License](https://img.shields.io/badge/license-MIT%2FApache--2.0-informational?style=flat-square)](COPYRIGHT.md)
[![standard-readme compliant](https://img.shields.io/badge/readme%20style-standard-brightgreen.svg?style=flat-square)](https://github.com/RichardLitt/standard-readme)

Octodo is a (work in progress) distributed Todo application.
The goal is to make an education distributed system that can be tinkered with.
This means an emphasis on architectural flexibility, with minimal complexity.
